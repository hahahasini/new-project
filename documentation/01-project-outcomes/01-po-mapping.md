## Overview

This document maps the PLACEHOLDER-PROJECT-NAME to the 12 standard Program Outcomes (PO1–PO12) defined by the National Board of Accreditation (NBA) for undergraduate engineering programs. Each mapping includes a justification level (High / Medium / Low) and a detailed explanation of how the project directly addresses the outcome through its technical implementation.

The Vitamin Deficiency Detection System is an AI-powered application that detects nutritional deficiencies from images of nails, tongues, and skin using custom-trained Convolutional Neural Networks (CNNs). It features a FastAPI backend serving three TensorFlow/Keras models, a React + Vite frontend with interactive data visualizations, personalized weekly diet plan generation, and a geolocation-based nearby doctors locator using OpenStreetMap and the Overpass API.
